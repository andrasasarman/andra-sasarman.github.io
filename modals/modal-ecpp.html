Tech Stack

  • WordPress Custom Theme Development(PHP + HTML) → built a custom theme with dedicated templates for every page
  • Dynamic Frontend powered by WordPress PHP API → all content is managed from the WP Admin (Custom Post Types + Meta Boxes) and rendered dynamically in PHP templates(clear separation of content vs. design)
  • Custom Post Types & Taxonomies → structured content
  • Advanced Meta Boxes → additional editable fields (text, images, repeaters for timelines) to extend content flexibility
  • Tailwind CSS + PostCSS (nested) → modern styling workflow, with automatic purge for optimized CSS output
  • Security Best Practices → implemented nonce verification, input sanitization, and output escaping
  • Performance & Scalability → modular architecture, easy to extend with new content types or sections
  • GiveWP (customized) → donation form plugin, integrated and adapted to match site design and workflow
  • WordPress AJAX API – implemented a custom Mailchimp subscription flow without page reload.
  • JavaScript (ES6) – for AJAX requests, dynamic UI, and form handling
